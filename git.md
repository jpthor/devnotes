## Git

# Setting up

Fork repo on website and jump in terminal:
```
git clone <url>
cd <file>
git checkout -b <branch>
git status
git add <file>
git commit -m <comment>
git push origin <branch>
```

Lifehack:
```
git add . && git commit -am "update" && git push origin master
```
